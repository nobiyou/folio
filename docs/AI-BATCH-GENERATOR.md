# AI批量生成器使用说明

## 功能概述

AI批量生成器可以自动为没有概述（摘要）和标签的文章批量生成内容，大大提高内容管理效率。

## 功能特点

- 🔍 智能扫描：自动扫描所有需要生成内容的文章
- 📝 批量生成：一键批量生成摘要和标签
- 📊 实时进度：显示生成进度和详细日志
- ⚙️ 灵活配置：可选择生成摘要、标签或两者
- 🎯 精准识别：自动识别缺少摘要或标签的文章

## 使用步骤

### 1. 配置AI API

在使用批量生成器之前，需要先配置AI API：

1. 进入 **外观 > 主题设置 > AI设置**
2. 配置至少一个AI API（支持OpenAI、Claude、DeepSeek等）
3. 测试连接确保API可用

### 2. 访问批量生成器

进入 **外观 > AI批量生成器**

### 3. 扫描文章

1. 选择文章类型（文章/页面）
2. 勾选需要生成的内容：
   - ✓ 没有摘要的文章
   - ✓ 没有标签的文章
3. 点击"扫描文章"按钮

### 4. 查看扫描结果

系统会显示所有需要生成内容的文章列表，包括：
- 文章标题
- 是否缺少摘要
- 是否缺少标签

### 5. 开始批量生成

1. 点击"开始批量生成"按钮
2. 系统会自动逐篇处理文章
3. 实时显示生成进度和日志
4. 完成后显示成功提示

## 生成规则

### 摘要生成

- 长度：100-150字
- 风格：专业、简洁、流畅
- 内容：提取文章核心要点
- 语言：根据主题设置自动选择中文或英文

### 标签生成

- 数量：5-8个关键词
- 类型：SEO友好的关键词
- 风格：自然、口语化
- 特点：
  - 写真类内容：自动提取人名
  - 商品类内容：突出产品特点
  - 避免重复标签

## 注意事项

1. **API配置**：确保AI API已正确配置且有足够的配额
2. **生成速度**：为避免API限流，每篇文章之间有1秒延迟
3. **内容质量**：生成后建议人工审核，确保内容准确性
4. **备份建议**：批量操作前建议备份数据库
5. **网络稳定**：确保网络连接稳定，避免生成中断

## 常见问题

### Q: 生成失败怎么办？

A: 检查以下几点：
- AI API配置是否正确
- API配额是否充足
- 网络连接是否稳定
- 查看错误日志获取详细信息

### Q: 可以中断生成吗？

A: 当前版本不支持中断，建议分批处理大量文章

### Q: 生成的内容可以修改吗？

A: 可以，生成后可以在文章编辑页面手动修改摘要和标签

### Q: 支持哪些文章类型？

A: 支持所有公开的文章类型（文章、页面等）
