## 🚀 快速开始

### 3步启用缓存
```bash
# 1. 确认环境
php -m | grep memcached

# 2. 访问管理页面
WordPress后台 → 工具 → 缓存管理

# 3. 一键安装
点击"安装Folio对象缓存"按钮
```

### 验证安装
```bash
# 运行测试脚本
访问: /wp-content/themes/folio/test-cache-management.php

# 查看健康状态
```
